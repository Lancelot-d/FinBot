# FinBot
LOG791 - Virtual assistant powered by Reddit data to provide personalized financial advice. Helps users make informed decisions on budgeting, saving, investing, taxes, and retirement planning.

# First thing to do after cloning 
- uv sync 
# Init with uv 
- uv venv venv --python=3.10
# Activate venv 
- .venv\Scripts\activate
# UV pip freeze 
- uv pip freeze > requirements.txt
# UV pip install 
- uv pip install -r requirements.txt
